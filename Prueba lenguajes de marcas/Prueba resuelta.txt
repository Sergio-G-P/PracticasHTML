Introducción
Para esta actividad he modelado mi curriculum utilizando HTML y CSS utilizando como plantilla una seleccionada por mí del grupo proporcionado por el profesor.
He utilizado programación en HTML para la estructura y CSS interno para el estilo para aproximarme lo más posible a la plantilla proporcionada. 

Ahora el desarrollo preciso.
Primero inicialice el documento e hice la head añadiendo las etiquetas meta esenciales tanto como el título de la página.

```
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Curriculum prueba Sergio Garcia Paricio</title>
</head>
<body>
</body>
</html>
```

Lo siguiente que hice fue cambiar el estilo de html y del body para crear una página y eliminar los márgenes por defecto.

```
	*{padding: 0px;margin: 0px;} /* Elimino los márgenes por defecto*/
        html{
            background-color: silver;
            font-family: sans-serif;
        }
        body{
            background: whitesmoke;
            width: 600px;
            margin: auto;
            min-height: 400px;
            display: flex;
        }
        /*Modificamos las características generales del cuerpo en general, color de fondo, anchura, márgenes y padding*/
```

A continuación dividí la página en 3, una división para el apartado de la izquierda y dos para el de la derecha, añadí las bases para su estilo, como color de fondo, márgenes, tamaño de fuente y color de fuente.

``` 
 	#izquierda{
            background: slategray;
            flex: 1;
            padding: 20px; 
            font-size: 11px;
            color: white;
        }
        /*Con el flex se divide la pagina en 3 secciones, 1 para la izquierda(flex 1) y 2 para la derecha(flex 2), 
        modificamos las características generales del texto y la distribución de esta sección*/
        #derecha{
            background: whitesmoke;
            flex: 2;
            padding: 20px;
            font-size: 12px;/*tamaño de las letras*/
        }
        /*Con el flex se divide la pagina en 3 secciones, 1 para la izquierda(flex 1) y 2 para la derecha(flex 2), 
        modificamos las características generales del texto y la distribución de esta sección*/
```

Lo siguiente fue comenzar a rellenar con información, comencé añadiendo una imagen con la etiqueta `<img>` y añadiendo dos `<article>` con listas `<ul>` dentro las que modifiqué con css para eliminar los puntos y los márgenes.

```
	#izquierda article ul{
            list-style-type: none;
            margin: 0px;
        }/*quito los puntos de las listas y el margen*/
```

Después comencé con el apartado derecha, con el nombre al que cambie el color con css `#nombre{color: slategray;}` y añadí una barra de separación horizontal `<hr>` a la que modifique con css para que pareciera a la plantilla `#nombre hr{height: 5px; background: black;}`.

Por último rellene el resto de la derecha utilizando ´<div>´ para dividir las secciones y utilizando `<h3>` como cabeceras de esas subdivisiones y párrafos `<p>` o listas `<ul>` para rellenar la información de esas subdivisiones a las que quite los puntos y márgenes.

```
 	#derecha div ul{
            list-style-type: none;
            margin: 0px;
        }/*quito los puntos de las listas y el margen*/
```
